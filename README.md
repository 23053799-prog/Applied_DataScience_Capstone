# Applied Data Science Capstone – SpaceX Launch Analysis

## Project Overview
This project analyzes SpaceX Falcon 9 launch data to understand launch patterns and predict first-stage landing success. The project follows a complete data science workflow from data collection to predictive modeling.

## Objectives
- Collect launch data using API and web scraping
- Perform data wrangling and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Build interactive visualizations and dashboards
- Develop a machine learning classification model to predict landing success

## Methodology

### 1. Data Collection
Data was collected using SpaceX API and web scraping techniques.

### 2. Data Wrangling
Data cleaning, transformation, and feature engineering were performed using Pandas.

### 3. Exploratory Data Analysis (EDA)
- SQL-based analysis
- Visualization using Matplotlib and Seaborn
- Interactive map visualization using Folium

### 4. Interactive Dashboard
Developed a dashboard using Plotly Dash to explore launch success trends.

### 5. Predictive Modeling
Built classification models to predict Falcon 9 first-stage landing success.
